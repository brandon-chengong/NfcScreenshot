# NfcScreenshot

刷门禁卡后自动截图 + 记录打卡时间的 Android 工具应用。

## 功能

- 监听 NFC 刷卡成功的通知，自动在 500ms 后截图
- 截图保存至相册 `Pictures/NfcScreenshots` 文件夹，文件名格式：`NFC_yyyyMMdd_HHmmss.png`
- 截图成功后推送通知
- 记录每次打卡时间，支持按日期查看历史记录及对应截图
- 自动计算每天首次/末次打卡时间与在场时长

## 使用方式

1. 安装 APK 后打开应用
2. 点击「前往开启无障碍权限」，找到「NFC刷卡截图」并开启
3. 返回应用，状态显示「✅ 无障碍服务已开启」即配置完成
4. 之后每次刷门禁卡，应用会自动截图并记录打卡时间

## 系统要求

- Android 11（API 30）及以上
- 需开启无障碍服务权限
- Android 13+ 需允许通知权限

## 触发关键词

当屏幕上出现以下文字时触发截图：

`已刷卡` / `刷卡成功` / `已读取` / `卡片已读`

## 构建

用 Android Studio 打开项目根目录，直接 Run 即可。

或手动触发 GitHub Actions 中的 **Build APK** workflow 获取编译产物。

## 权限说明

| 权限 | 用途 |
|------|------|
| 无障碍服务 | 监听屏幕 NFC 刷卡通知 |
| 截图（系统 API）| 调用 `takeScreenshot` 截取当前屏幕 |
| 写入外部存储 | 保存截图到相册 |
| 发送通知 | 截图成功后提醒 |
